# AgentMemorySystem 依赖
# ========================
#
# 推荐：下载 AgentMemorySync.exe（已内置所有依赖，无需安装）
#       https://github.com/LEE20260315/AgentMemorySystem/releases

# 源码运行：
#   安装全部依赖:  pip install -r requirements.txt

#
# 打包为 EXE:    python build.py

# --- 核心依赖（必须）---
